# exact-workshop-demo

A Spring Boot demo application built for workshop purposes.

## Tech Stack

- Java 21
- Spring Boot 3.5.3
- Spring Web
- Gradle

## Getting Started

### Run the application

```bash
./gradlew bootRun
```

### Run tests

```bash
./gradlew test
```

### Build

```bash
./gradlew build
```

The application starts on `http://localhost:8080` by default.
